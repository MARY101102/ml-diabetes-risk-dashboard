# Machine Learning-Based Diabetes Risk Estimation Dashboard

## Project Overview

This project is a full-stack machine learning web application that estimates
diabetes-related risk indicators using health and lifestyle input data.

The system will include:

- Machine learning classification model
- React dashboard
- FastAPI backend
- User authentication
- Assessment history
- PDF report generation
- Explainable AI results
- Dashboard analytics
- Cloud deployment

## Important Disclaimer

This application is developed for educational and portfolio purposes only.
It does not provide a medical diagnosis, medical advice, or treatment
recommendations.

## Dataset

The project uses the CDC Diabetes Health Indicators dataset from the
UCI Machine Learning Repository.

Target variable:

- `0` = No diabetes
- `1` = Prediabetes or diabetes
